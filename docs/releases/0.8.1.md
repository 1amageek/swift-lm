# swift-lm 0.8.1

`0.8.1` is a dependency-refresh patch release for `0.8.0`. It keeps the
real-model output correctness fixes from `0.8.0` and updates Swift package
dependencies to their latest available tags verified on April 25, 2026.

## What 0.8.1 Changes

### Dependency updates

Direct dependencies:

- `swift-collections` `1.3.0` -> `1.4.1`
- `swift-jinja` `2.3.2` -> `2.3.5`
- `swift-transformers` `1.1.9` -> `1.3.0`
- `JSONSchema` remains `1.3.1`
- `swift-testing-heartbeat` remains `0.1.0`

Transitive dependencies now resolve to:

- `swift-huggingface` `0.9.0`
- `swift-crypto` `4.5.0`
- `swift-asn1` `1.7.0`
- `swift-nio` `2.99.0`
- `swift-atomics` `1.3.0`
- `swift-system` `1.6.4`
- `EventSource` `1.4.1`
- `yyjson` `0.12.0`

### Package constraint update

`swift-transformers` now uses a normal `from: "1.3.0"` constraint instead of
being pinned to the `1.1.x` minor line via `.upToNextMinor(from: "1.1.9")`.

## Correctness

Validated before release:

```bash
swift package update
perl -e 'alarm shift; exec @ARGV' 120 xcodebuild build-for-testing -scheme swift-lm-Package -destination 'platform=macOS'
perl -e 'alarm shift; exec @ARGV' 120 xcodebuild test-without-building -scheme swift-lm-Package -destination 'platform=macOS' -only-testing:'SwiftLMTests/RealOutputEvidenceTests'
perl -e 'alarm shift; exec @ARGV' 120 xcodebuild test-without-building -scheme swift-lm-Package -destination 'platform=macOS' -only-testing:'MetalCompilerTests/MetalSourceGeneratorTests/ssmRecurrenceUsesFloatUnitOffsetNormWeight()' -only-testing:'MetalCompilerTests/MetalSourceGeneratorTests/batchedQKNormDecodeAppliesWeightBias()'
```

Observed real-model outputs remain correct:

- Qwen3.5: `Japan's capital is Tokyo,`
- LFM: `Tokyo. However, the user`
- Gemma4: `Tokyo`

## Notes

- `swift-transformers 1.3.0` brings in `swift-huggingface 0.9.0`, which can
  print Contacts/CoreData sandbox diagnostics during local XCTest runs. The
  diagnostics did not affect build or output-correctness test results.
- No public `SwiftLM` API migration is required.

dependencies: [
